# 扫雷游戏 (Minesweeper)

这是一个使用 Java (Swing/AWT) 开发的经典扫雷游戏。本项目实现了扫雷的核心玩法，并包含了多种难度选择及自定义模式。

## ✨ 主要功能

- **多级难度**：
  - 简单 (Simple)
  - 普通 (Normal)
  - 困难 (Hard)
  - **自定义 (Custom)**：允许玩家自行设置地图大小和雷数。
- **核心玩法**：
  - 左键：翻开方块。
  - 右键：插旗/取消插旗。
  - **双击/数字右键**：当某一数字周围的旗帜数量符合要求时，右键点击该数字可快速翻开周围剩余的方块（若旗帜位置错误则判负）。
- **游戏控制**：
  - 点击顶部中央图标：快速重置当前关卡。
  - 鼠标中键：返回主菜单（难度选择界面）。
- **计时器**：记录每局游戏的时间。

## 🛠️ 环境要求

- **JDK 版本**：JDK 1.8 及以上

## 🚀 如何运行

### 方式 1：运行打包好的 JAR 文件
如果目录下已有 `minesweep.jar`，可以直接双击运行，或在命令行中输入：
```bash
java -jar minesweep.jar
```

### 方式 2：从源码编译运行
1. 确保安装了 JDK 1.8+。
2. 进入 `minesweep/src` 目录。
3. 编译源码：
   ```bash
   javac Gamewin.java
   ```
4. 运行主程序：
   ```bash
   java Gamewin
   ```
   *(注：具体命令可能需根据包结构调整，建议使用 IntelliJ IDEA 或 Eclipse 导入项目运行)*

## 🎮 操作说明

| 操作 | 功能 |
| --- | --- |
| **左键单击** | 翻开未探测的方块 |
| **右键单击** | 标记/取消标记地雷 (插旗) |
| **右键单击数字** | 当周围旗帜数等于该数字时，快速翻开周围未标记方块 |
| **鼠标中键** | 随时返回主菜单 |
| **笑脸图标** | 重置当前游戏 |
